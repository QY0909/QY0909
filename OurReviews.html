OUR REVIEWS

I'm glad to have this opportunity to learn a new keyboard layout, Colemak, and practise typing with it. Since I'm used to the QWERTY keyboard layout, it is difficult for me to practise using this new layout at first. However, I would say that Colemak is actually a beginner-friendly keyboard layout as it is designed based on the most frequently used letters in our daily life. If we have more time to practise the Colemak keyboard layout, I believe it will result in smoother and more efficient typing. In my opinion, the Colemak keyboard layout should be introduced to more keyboard users around the world.

-Goh

In my opinion, learning colemak is similar to gaining more general knowledge. It's interesting to experiment with new things, as when I used Colemak for seven days. While at first it was challenging, for me it is typical,because I now understand how the letters are arranged in Colemak. Because the most often used letters are close to our fingertips, clicking other letters is quick, but because I'm not moving my hands, my wrists begin to ache. Therefore, even though I have to move my hands, I prefer the QWERTY keyboard because I've been using it for years without experiencing any hand pain.

-Narmatha

I learn Colemak on a website called 'Colemak Club'. On the website, there are many levels. Each level will get harder. I practiced the Colemak keyboard for 1 week on the website. On the first day, it was hard for me to use Colemak keyboard, but on the following days I can catch up with it. On the Colemak keyboard, the most common letters occupy the home row,so it might be easier for new users who want to use the keyboard for the first time. Even though I can catch up with Colemak ,I still get confused about the keys to QWERTY keys sometimes. I would say that the experience for me when learning Colemak was fun. For me, I still prefer QWERTY even though I easily catch up with Colemak because I have used the QWERTY keyboard since an early age. Comparing between QWERTY and Colemak, QWERTY is the one that works for me.

-Navaniswary

When I first learn new keyboard, it's really hard actually because I already used with QWERTY keyboard arrangement. After learning for 15 minutes each day, I had discovered some advantages and disadvantages in using Colemak keyboard. The most obvious one is Colemak changed how long I'm able to type without feeling pain on my fingertips. However, the possibility to get typewriter jamming is higher. If I have to choose one keyboard for daily use, I would prefer QWERTY. However, If I had a chance to learn Colemak keyboard for a longer period, my preference might change in the future.

-Farisha
